# **App Name**: iGarage

## Core Features:

- Landing Page: Displays a professional landing page with clear sections for hero, about, services, and testimonials.
- Service Catalog: Lists available car maintenance and repair services with detailed descriptions.
- Customer Testimonials: Displays customer testimonials to build trust and credibility.
- Contact Information: Provides contact details, including phone and WhatsApp links, address, and a Google Map integration.
- Booking Form: A form that allows users to request service and provide information for appointment scheduling.
- Service Duration Estimation: Estimate service duration times using an LLM as a tool

## Style Guidelines:

- Primary color: Deep blue (#1A4397) to evoke trust and professionalism.
- Background color: Light gray (#E9ECEF) for a clean and modern look.
- Accent color: Gold (#FFD700) for highlights and CTAs, representing value and premium service.
- Body text and headlines: 'Inter', a grotesque-style sans-serif for a modern, neutral look.
- Use Font Awesome icons for services and contact information to enhance visual appeal.
- A clean, grid-based layout for easy navigation and content organization.
- Subtle transitions and animations for interactive elements and form submissions.
